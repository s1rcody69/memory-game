#  Africa Memory Match

**Africa Memory Match** is an interactive, high-performance educational game designed to celebrate African culture while challenging the player's cognitive skills.

This project was developed as a **Capstone Project using React.js and Modern CSS.**

---

##  Live Demo

**Experience the live game here:** 🚀 [**Africa Memory Match on Vercel**](https://s1rcody69-memory-game-five.vercel.app)

---

##  The Problem

Many memory games are generic and lack cultural depth or personalized user experiences. Standard web games often lack:
- **Persistence:** Progress is lost after refreshing the browser.
- **Personalization:** No way to track individual growth, high scores, or streaks.
- **Engagement:** Static visuals that don't adapt to user skill levels or interests.

---

##  The Solution

Africa Memory Match provides a **dynamic platform** where users can:
- Explore African geography, wildlife, and nature through high-quality imagery and CDNs.
- Save progress to a localized leaderboard and compete for the top rank.
- Build daily habits through a logic-driven streak-tracking system.
- Experience tactile, responsive gameplay with hardware-accelerated 3D card physics.

---

##  Key Features

- **Thematic Variety:** Toggle between African Flags, Wildlife, and Tropical Fruits.
- **3-Tier Difficulty:** Customizable grid sizes (Easy, Medium, Hard) to suit all skill levels.
- **Authentication System:** Secure local login and registration system with password hashing.
- **Global Leaderboard:** Real-time ranking that highlights your personal best against others.
- **Streak System:** Rewards daily consistency with "Flame" milestones (3, 7, 14, and 30 days).
- **Glassmorphic UI:** A modern, semi-transparent design optimized for seamless play on mobile and desktop.

---

##  Difficulty Configuration

The game uses a tiered system to scale the challenge and grid density:

| Level | Pairs | Layout | Game Logic |
|:--- |:--- |:--- |:--- |
|  **Easy** | 6 | 3x4 Grid | Ideal for beginners and children |
|  **Medium** | 8 | 4x4 Grid | Standard competitive challenge |
| **Hard** | 12 | 6x4 Grid | Expert level concentration required |

---

##  Technology Stack

This project was built using a modern frontend stack:

* **React.js** – Component-based architecture and custom hooks (`useGameLogic`).
* **Lucide React** – Clean, scalable vector iconography.
* **CSS3** – 3D transforms, backdrop filters, and custom keyframe animations.
* **LocalStorage API** – For persistent user data, settings, and score tracking.

---

## Target Users

Africa Memory Match is designed for:

* **Students:** A fun way to learn and memorize African country flags and animal names.
* **Casual Gamers:** People looking for quick, aesthetic brain-teasing sessions.
* **Web Developers:** A reference for implementing high-performance animations and custom hooks in React.
